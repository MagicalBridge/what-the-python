# What The Python

Python 基础入门学习笔记，跟随尚硅谷 Python 教程编写的练习代码。

## 项目简介

本项目是一个 Python 入门学习仓库，按章节组织代码示例与练习，涵盖从基础语法到流程控制的核心知识点。每个 `.py` 文件对应一个独立的知识点或练习，可直接运行查看效果。

## 学习内容

| 章节 | 主题 | 知识点 |
|------|------|--------|
| 02 | Python 入门语法 | 字面量、注释、变量、数据类型、类型转换、标识符、运算符、字符串操作、格式化输出、数据输入 |
| 03 | 判断语句 | 布尔类型、比较运算符、if / if-else / if-elif-else、嵌套判断 |
| 04 | 循环语句 | while 循环、累加求和、猜数字游戏 |
